# dados (jeito simples)

1. baixa o dataset no kaggle.
2. coloca o arquivo excel nesta pasta:
`data/raw/Data Model IoTMLCQ 2024.xlsx`

se quiser baixar pelo terminal:

```bash
kaggle datasets download -d jocelyndumlao/iot-monitoring-of-water-quality-and-tilapia -p data/raw --unzip
```

se o nome do arquivo vier diferente, tudo bem.
e so passar o caminho certo no `--data` quando rodar os scripts.
